# 函数图像绘制工具

这是一个简洁高效的函数图像绘制工具，可以将坐标点转换为可视化的函数图像。

## 功能特点

1. 坐标输入功能
   - 可自定义横坐标的起始值、终止值和间隔
   - 支持一个横坐标对应一个或两个纵坐标
   - 支持表格形式输入数据

2. 图像显示功能
   - 支持折线图显示
   - 支持散点图显示
   - 支持图像样式自定义

3. 界面特点
   - 简洁现代的设计风格
   - 响应式布局，适配各种设备
   - 直观的操作界面

## 技术实现
- 使用HTML5和CSS3构建界面
- 使用Chart.js库实现图表绘制
- 采用Flexbox布局实现响应式设计
